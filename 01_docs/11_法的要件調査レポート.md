## 1. 概要

### 1.1 サービスの特性

uketsuguAIは以下の法的に重要な特性を持つサービスです：

| 特性 | 法的影響 |
|------|---------|
| **個人情報の取得・処理** | 個人情報保護法の適用 |
| **決済機能（Stripe）** | 特定商取引法・資金決済法の適用 |
| **メッセージング機能** | 電気通信事業法の適用可能性 |
| **AI/LLM利用** | AI事業者ガイドラインの適用 |
| **死亡・相続関連情報提供** | 弁護士法（非弁行為）への配慮 |
| **LINE公式アカウント** | LINEプラットフォーム利用規約への準拠 |
| **センシティブ情報の取扱い** | 要配慮個人情報への配慮 |

### 1.2 リスク評価

**高リスク領域:**
- 個人情報漏洩（遺族情報、故人情報）
- 非弁行為への抵触
- 消費者契約法違反（不当条項）

**中リスク領域:**
- 特定商取引法違反（表示義務）
- 電気通信事業法違反（外部送信規律）
- AI利用に関する免責不足

## 3. 個人情報保護法への対応

### 3.1 プライバシーポリシーに記載すべき事項

**基本事項（法定記載事項）:**

```
1. 事業者の名称・住所・代表者名
2. 個人情報保護管理者の連絡先
3. 取得する個人情報の項目
   - LINE User ID
   - 表示名
   - 故人との続柄
   - 故人の死亡日
   - 居住地（都道府県・市区町村）
   - 会話履歴
   - タスク進捗情報
   - 決済情報（Stripe経由）

4. 利用目的の具体的明示
   - タスク生成・管理のため
   - リマインダー送信のため
   - サービス改善のため
   - Pay It Forward機能のため

5. 第三者提供に関する事項
   - 提供先: Google Cloud (Gemini API)、OpenAI、Stripe、LINE
   - 提供目的: AI処理、決済処理、メッセージ送信
   - 匿名化処理の実施内容

6. 開示・訂正・削除請求への対応手続き
7. 安全管理措置の内容
   - 技術的措置（暗号化、アクセス制御）
   - 組織的措置（アクセス権限管理）
   - 物理的措置（Cloud SQL利用）
8. Cookie・類似技術の使用に関する説明
```

### 3.2 2024年4月施行の改正法対応

**重要変更点:**

1. **漏えい等報告義務の拡大**
   - 委託先での漏えいも報告対象
   - 個人情報（個人データに限らない）も対象
   - 報告期限: 発覚後速やかに（概ね3〜5日以内）

2. **Webスキミング対策**
   - 外部からの不正アクセスによる情報漏洩への対策
   - セキュリティ監視の強化

3. **安全管理措置の明確化**
   - 具体的な技術対策の記載
   - 委託先管理の明示

### 3.3 データ保持・削除ポリシー

**推奨設定（現在の実装を明文化）:**

```
- 会話履歴: 1ヶ月保持後自動削除
- ユーザープロフィール: 退会後90日で完全削除
- 決済情報: 法定保存期間（7年）後削除
- タスク情報: ユーザー希望に基づき削除
- バックアップ: 30日保持
```

### 3.4 要配慮個人情報への配慮

uketsuguAIは死亡情報を扱うため、以下に注意:

- 故人の死因は取得しない（取得する場合は要配慮個人情報）
- 宗教・信条に関する情報は取得しない
- 取得する場合は明示的同意が必要

### 5.1 外部送信規律（2023年6月施行）

**uketsuguAIの該当性:**

LINE公式アカウントを通じたメッセージングサービスは、「他人の通信を媒介するサービス」に該当する可能性があります。

**必要な対応:**

1. **外部送信に関する公表事項**

```markdown
# 外部送信規律に基づく情報公開

## 送信される情報と送信先

### 1. LINE Corporation
- 送信される情報: ユーザーID、メッセージ内容、タイムスタンプ
- 送信先: LINE株式会社
- 利用目的: メッセージ送受信のため

### 2. Google Cloud (Gemini API)
- 送信される情報: 匿名化されたユーザー情報、会話内容
- 送信先: Google LLC
- 利用目的: AI応答生成のため

### 3. OpenAI
- 送信される情報: テキストデータ（Embedding処理用）
- 送信先: OpenAI, Inc.
- 利用目的: ベクトル化処理のため

### 4. Stripe
- 送信される情報: 決済情報
- 送信先: Stripe, Inc.
- 利用目的: 決済処理のため

### 5. Google Analytics（使用する場合）
- 送信される情報: 利用状況データ
- 送信先: Google LLC
- 利用目的: サービス分析のため
```

- 初回利用時にポップアップで通知

## 6. 消費者契約法への対応

### 6.1 避けるべき不当条項

**無効となる条項の例:**

```
❌ 「いかなる場合も当社は一切の責任を負いません」
   → サルベージ条項として無効

❌ 「当社の判断により、予告なくアカウントを削除できます」
   → 消費者の利益を一方的に害する条項

❌ 「東京地方裁判所を専属的合意管轄とします」
   → 消費者の裁判を受ける権利を制限
```

**適切な条項の例:**

```
✅ 「当社の故意または重過失による場合を除き、
    損害賠償額は過去12ヶ月間にお支払いいただいた金額を上限とします」

✅ 「利用規約に違反した場合、事前通知の上、
    是正期間を設けた後にアカウントを停止することがあります」

✅ 「第一審の専属的合意管轄裁判所を東京地方裁判所とします。
    ただし、消費者契約法の適用がある場合はこの限りではありません」
```

### 6.2 2023年6月改正（令和4年改正）への対応

- **サルベージ条項の禁止**: 免責範囲を明確化
- **解除権行使の妨害禁止**: 退会手続きを分かりやすく
- **努力義務の追加**: 契約条件の情報提供を丁寧に

## 7. AI・生成AI関連規制への対応

### 7.1 AI事業者ガイドライン（2024年4月）

**遵守すべき原則:**

1. **人間中心**: AIは人間の判断を支援するツール
2. **安全性**: ユーザーへの害を最小化
3. **公平性**: 差別的な出力を避ける
4. **透明性**: AI利用を明示
5. **説明責任**: 不適切な出力への対応

### 7.2 利用規約・プライバシーポリシーでの明示事項

```markdown
## AI利用に関する重要事項

### 1. AI技術の利用
本サービスでは、Google Gemini等の生成AIを利用してタスク提案を行っています。

### 2. AI生成コンテンツの性質
- AIによる提案は参考情報であり、法的助言ではありません
- 情報の正確性は保証されません
- 重要な判断は専門家（弁護士、行政書士等）にご相談ください

### 3. データの取扱い
- AIへの送信時、個人情報は匿名化・一般化処理を行います
- 市区町村情報は地域ブロックに変換
- 死亡日は経過期間に変換
- AIサービス提供元の利用規約に従います

### 4. 学習データへの利用
お客様の会話データは、AIモデルの再学習には使用しません。
ただし、サービス改善のための統計分析には匿名化して使用する場合があります。

### 5. 著作権
AI生成コンテンツの著作権は当社に帰属します。
```

### 7.3 個人情報保護委員会の注意喚起への対応

- プロンプトに個人情報を含めない（匿名化処理を実施済み）
- 機密情報の入力を避けるよう注意喚起
- AI出力の二次利用に関する規定

## 8. 弁護士法（非弁行為）への対応

### 8.1 非弁行為のリスク

**弁護士法72条:**
弁護士でない者が、報酬を得る目的で、法律事件に関する法律事務を業として行うことは禁止。

**uketsuguAIでの注意点:**
- 遺産分割の具体的なアドバイス → 非弁行為
- 相続手続きの法的判断 → 非弁行為
- 遺言書作成の具体的指導 → 非弁行為（司法書士・行政書士も限定的）

### 8.2 サービス設計での対策（既に実装済みを明文化）

**利用規約での明示:**

```markdown
## サービスの範囲と限界

### 1. 本サービスが提供するもの
- 死亡後の手続きに関する一般的な情報提供
- タスクの整理と進捗管理
- 補助金・給付金に関する情報提供
- 期限や優先順位の提示

### 2. 本サービスが提供しないもの（重要）
- 法律相談・法的アドバイス
- 遺産分割に関する具体的な助言
- 税務相談・税務申告の代行
- 手続きの代行
- 専門家としての判断

### 3. 専門家への相談推奨
以下の場合は、必ず専門家にご相談ください：
- 相続財産に関する争いがある場合
- 遺言書の有効性に疑問がある場合
- 相続税の申告が必要な場合
- 不動産の名義変更が必要な場合

専門家の例：
- 弁護士（法的紛争、遺産分割協議）
- 司法書士（不動産登記、遺言書検認）
- 行政書士（各種届出書類作成）
- 税理士（相続税申告）
- 社会保険労務士（年金手続き）
```

### 8.3 免責条項

```markdown
## 免責事項

本サービスは情報提供およびタスク管理支援を目的としており、
法律、税務、その他の専門的アドバイスを提供するものではありません。

本サービスの利用により得られた情報に基づいて行動される場合は、
必ず関連する専門家にご相談の上、ご自身の責任において判断してください。

当社は、本サービスの情報の正確性、完全性、有用性について
いかなる保証も行いません。
```

## 禁止事項

   - サービスを転売する行為

7. **その他**
   - 反社会的勢力への利益供与
   - 当社が不適切と判断する行為

## 10. LINEプラットフォーム固有の要件

### 10.1 LINE公式アカウント利用規約への準拠

**必須事項:**

**個人情報漏洩時の対応:**

2. 本人への通知
3. 二次被害防止措置
